intersectionSafe is simple intersection code, iterates along two sorted arrays incrementing the index of the smaller value (if values are equal, it is part of intersection, and so store position and iterate both).   • [Object]
)
Integer

Defined in build/js/j-visualizer.js:1199

waitForRepeatingEvents is a debounce-like function for preventing multiple execution. It should be called with a function and an amount of time to wait. It can also be provided a timer name so as to namespace the blocked items. It can also be called with an Object timer set to allow different objects to not prevent the execution of functions in others.

Parameters:

  • callback Function

    The function to be called at end of waiting time

  • timeout Integer

    Number of milliseconds to wait before exectuing function

  • [String] Object optional

    timerName A namespace for the debounce @default "default timer"

  • [Object] Object optional

    timerSet A collection of timers to use in blocking functions @default

Returns:

Integer:

The Timeout id assigned by the browser for the created timeout.
